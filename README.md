# Asistente de Diagnóstico Predictivo HVAC con IA ⚙️❄️

Este proyecto fue desarrollado como una solución integral aplicando principios de Ingeniería Mecánica Eléctrica y automatización con inteligencia artificial. El sistema automatiza la evaluación termodinámica de sistemas de climatización y refrigeración por medio de la API de Google Gemini.

Está diseñado como una herramienta de apoyo técnico para la industria del mantenimiento de aires acondicionados, agilizando la toma de decisiones en campo.

## Características Principales
* **Análisis Termodinámico:** Interpreta la relación entre la presión de succión, amperaje, temperatura ambiente y temperatura del evaporador.
* **Auto-detección de Modelo:** Se conecta dinámicamente a la versión más estable y reciente del modelo Gemini Flash.
* **Reportes Automatizados:** Genera un diagnóstico técnico estructurado, señalando posibles fallas (ej. falta de refrigerante R-410A o problemas en el serpentín) y emitiendo recomendaciones de mantenimiento.

## Instalación y Ejecución
1. Clona este repositorio: `git clone https://github.com/tu_usuario/hvac-diagnostics-ai.git`
2. Instala la dependencia oficial de Google: `pip install google-generativeai`
3. Ejecuta el programa principal: `python diagnostico_hvac.py`
4. Ingresa tu API Key de Google Gemini cuando el sistema lo solicite.
